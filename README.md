# Financial Performance & Profitability Dashboard

## 📌 Project Overview

This project presents an interactive Financial Performance Dashboard developed using Power BI to analyze revenue trends, profitability, forecasting, and overall business performance.

## 📊 Features

* Revenue and profit analysis
* Financial KPI monitoring
* Profitability trend analysis
* Discount impact analysis
* Forecasting visualization
* Interactive filters and slicers

## 🛠 Tools Used

* Power BI
* DAX
* Data Visualization Techniques

## 📈 Key Insights

* Revenue showed a strong upward trend over time.
* Higher discount bands negatively impacted profitability.
* Government segment contributed the highest revenue.
